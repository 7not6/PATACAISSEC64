# PATACAISSE C64
<br/>
PATACAISSE C64 is a Commodore 64 RushHour like game,<br/>
based on PATACAISSE VG5000 by BDCIRON.

OSCAR64 compiler https://github.com/drmortalwombat/oscar64

```oscar64 -n patac64.c```
